# Event Estimator Website

🚀 **Live Demo:** https://event-estimator-website.vercel.app/

## Overview

Event Estimator Website is a responsive web application that helps users estimate and manage event-related expenses efficiently. The platform provides an intuitive interface for planning events, calculating estimated costs, and organizing budget information.

This project was developed to demonstrate front-end web development skills using HTML, CSS, and JavaScript while focusing on usability, responsiveness, and clean design.

## Features

* Interactive event cost estimation
* User-friendly and responsive interface
* Real-time calculation of estimated expenses
* Clean and modern design
* Mobile and desktop compatibility
* Fast and lightweight performance

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Vercel (Deployment)
